核心功能

专业水上运动工具箱

从实时 GPS 到视频录制，水域覆盖你水上运动的每一个需求

实时 GPS 轨迹

高德 SDK 高精度定位，0.8 秒刷新一次，水面上稳定追踪。息屏后前台服务保活，长时间划行不丢轨迹。

  • 高精度 GPS 优先模式
  • 息屏持续追踪（前台服务）
  • 卫星图 / 标准图切换
  • 实时航向与坐标显示

运动数据统计

实时显示当前速度、最高速度、平均速度、已划距离、海拔高度和航向角，运动全程数据一览无余。

  • 速度 / 最高速度 / 均速
  • 总距离与海拔
  • 运动时长计时
  • 每公里语音里程播报

路书规划

预先规划水上路线，运动中实时偏航提醒。支持多航点设置，沿途自动计算与计划路线的偏差距离。

  • 多航点路线设置
  • 实时偏航距离提示
  • 路线进度追踪
  • 历史路线收藏复用

同步视频录制

运动同时录制视频，支持前后摄像头随时切换。视频自动保存至手机相册，运动轨迹与画面完美留存。

  • 运动同步视频录制
  • 前后摄像头切换
  • 自动保存至相册
  • 视频小窗叠加地图

下水点导航

记录常用下水点，一键唤起高德 / 百度 / 腾讯地图导航，未安装导航 APP 时自动回退网页地图。

  • 常用下水点收藏
  • 高德 / 百度 / 腾讯导航
  • 网页地图兜底
  • 附近水域发现

历史活动记录

每次运动自动保存为活动记录，包含完整轨迹、时间、距离、速度等数据，随时回顾运动历程。

  • 完整轨迹回放
  • 运动数据归档
  • 按日期筛选查看
  • 数据本地安全存储

天气与风速

出发前查看当前位置天气，实时风速与风向显示在运动界面，帮助水上运动者提前评估水面条件。

  • 实时风速 / 风向
  • 彩云天气接入
  • 运动界面即时显示

TTS 里程播报

每划行满 1 公里，自动播报已划里程、运动时长和当前速度。息屏状态下正常工作，解放双眼专注划行。

  • 每公里自动语音播报
  • 播报里程 / 时长 / 速度
  • 息屏后正常运行
  • 可开关广播功能
